Toolbar Example

Accessibility Features

  • The role=toolbar on the div element containing the button elements identifies the toolbar.
  • The aria-controls attribute on the button elements idenfities that the buttons alter the text area.
  • All but one button element have tabindex="-1" to remove them from tab order.
  • The button elements onClick handles altering the text area.
  • The role = "group" on the div element identifies that the following button elements are related

Keyboard Support

Toolbar Buttons

Key Function
Left Arrow Key The button to the left of the current button is focused and put into tab order in place of the previous button.
Right Arrow Key The button to the right of the current button is focused and put into tab order in place of the previous button.
Return Key Activate the selected button
