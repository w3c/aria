Accessibility Features

  • The role=tabpanel on the div element identifies the tabpanel.
  • The aria-controls attribute on the li[role="tab"] elements idenfities them as tabs.
  • All but one li[role="tab"] element have tabindex="-1" to remove them from tab order.
  • The li[role="tab"] elements have event handlers to change tabs.
  • The role = "group" on the div element identifies that the following button elements are related

Keyboard Support

Key Function
Up Arrow Key The previous tab is focused and put into tab-order in place of the current tab.
Down Arrow Key The next tab is focused and put into tab-order in place of the current tab.
Return Key and Space Bar Activate the selected tab
